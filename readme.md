# Aplikasi MVC Mahasiswa

## Kelompok

- Anggota 1 : Backend
- Anggota 2 : Frontend
- Anggota 3 : Dokumentasi

## Arsitektur

Aplikasi menggunakan pola MVC:

- Model
- View
- Controller

## Cara Menjalankan

1. Clone repository
2. Pindahkan ke xampp/htdocs
3. Import database
4. Jalankan Apache dan MySQL
5. Buka:

http://localhost/mvc_mahasiswa/public

## Fitur

- CRUD Mahasiswa
- Search
- Filter
- Export CSV
- Export PDF
- Bootstrap Responsive
